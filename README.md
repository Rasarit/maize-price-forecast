# Maize Price Forecast using ARIMA (R)

This project analyzes maize mandi price data and forecasts future prices using time series modeling.

## Dataset
Source: AGMARKNET mandi price data

## Tools Used
- R
- ggplot2
- forecast package

## Steps
1. Data cleaning
2. Price trend visualization
3. Time series conversion
4. ARIMA model fitting
5. Price forecasting for next 12 months

## Output
Forecasted maize prices based on historical mandi data.
